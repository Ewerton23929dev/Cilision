# Cilision project

> Um projeto que desenvolve um interface para o C em Python!

## Funcionalidades!
- **Montagem de codigo C**:  Permitir aos devs gerar codigo C de forma simples em python!
- **Suporte a varios metodos!**:  A bliblioteca tem suporte para diversas metodos do C como malloc, free e diversas!
- **Facilidade!**: Utilize modulos e blocos para montar seu codigo C em python! 

## Baixar!
1. Para baixar basta ter o python na maquina e clonar o repositorio.
```bash
    git clone https://github.com/Ewerton23929dev/Cilision.git
```
2. Testar e aproveitar!
